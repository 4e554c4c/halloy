# `[font]`

 Application wide font settings.

**Example**

```toml
[font]
family = "Comic Mono"
size = 13
```

## `family`

Monospaced font family to use.

- **type**: string
- **values**: any string
- **default**: not set [^1]

## `size`

Font size.

- **type**: integer
- **values**: any positive integer
- **default**: 13

[^1]: Iosevka Term is provided by the application, and used by default.
