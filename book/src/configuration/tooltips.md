# Tooltips

## `[tooltips]` Section

```toml
button = true | false
user = true | false
```

| Key      | Description                                       | Default |
| -------- | ------------------------------------------------- | ------- |
| `button` | Control if tooltips are displayed for buttons.    | `true`  |
| `user`   | Control if tooltips are displayed for user names. | `true`  |
