# `[font]`

Application wide font settings.

## `family`

Monospaced font family to use.

```toml
# Type: string
# Values: any string
# Default: not set
#
# Note: Iosevka Term is provided by the application, and used by default.

[font]
family = "Comic Mono"
```

## `size`

Font size.

```toml
# Type: integer
# Values: any positive integer
# Default: 13

[font]
size = 13
```
